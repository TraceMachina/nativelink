---
title: Is NativeLink free?
description: The open-source core is free forever. Cloud and Enterprise are paid offerings.
---

**Yes — the open-source release is MIT-licensed and free forever.** You can
self-host it on any infrastructure you want, modify it, redistribute it.

If you want us to run it for you, there are two paid tiers:

- **Cloud** — managed multi-tenant cluster, $1,000/month flat.
- **Enterprise** — single-tenant, on-prem, custom contracts.

See the [pricing page](https://nativelink.com/pricing) for the full breakdown.

---
title: What is LRE?
description: Local Remote Execution — hermetic builds on your machine without containers.
---

LRE (Local Remote Execution) lets you run remote-execution-style builds
locally — same hermeticity guarantees, same content-addressed inputs, but
without shipping the action over the network.

The full write-up lives in [Explanations → LRE](/docs/explanations/lre).

NativeLink is a high-performance remote build cache and execution platform for Bazel,
Buck2, Reclient, and other build systems that speak the Remote Execution API.

This site is a Phase 1 scaffold. Content will be migrated from the existing
docs at `web/platform/src/content/docs/` in a later commit.

## Why NativeLink

- Sub-millisecond cache hits backed by a distributed CAS layer.
- Local Remote Execution that reuses your local toolchains without containers.
- A single JSON5 file controls every store and service.

Hands-on examples for the most common remote-execution patterns:

- **Cache-only** — every action runs locally, results are stored remotely.
- **Cache + execution** — actions ship to the worker fleet; outputs come
  back.
- **Hybrid** — local for fast actions, remote for everything else, driven by
  policy.

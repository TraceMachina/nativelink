What a production-grade NativeLink cluster actually looks like:

- Sharded CAS backed by S3 + Redis for hot-key absorption.
- Multi-region scheduler with deterministic worker assignment.
- Autoscaling worker pools driven by queue depth.
- Prometheus / OpenTelemetry pipelines.
- mTLS between every hop; signed action results.

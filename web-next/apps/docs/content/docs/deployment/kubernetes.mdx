A reference NativeLink deployment on Kubernetes:

- StatefulSet for the scheduler + CAS.
- Deployment for the action-cache reader replicas.
- HPA-driven worker pool with cluster-autoscaler integration.
- Optional: external storage class for warm CAS tiers.

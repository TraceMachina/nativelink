NativeLink emits OpenTelemetry traces and Prometheus metrics out of the box:

- Per-action latency histograms.
- Cache-hit ratio by store.
- Worker queue depth and dispatch latency.
- Per-cluster connection counts.

This page shows a working Grafana dashboard and the Prometheus scrape config

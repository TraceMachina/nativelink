NativeLink core is free open source software under the
[Apache 2.0 license](https://github.com/TraceMachina/nativelink/blob/main/LICENSE).

NativeLink Cloud is currently free for users and organizations, with some
limits on the storage of build artifacts.

If you have questions about unique cloud workloads or would like to contract
assistance with on-prem setups, please reach out in our
[Slack](https://forms.gle/LtaWSixEC6bYi5xF7).

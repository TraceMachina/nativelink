NativeLink speaks the standard Remote Execution API. If your build tool does
too, you can swap NativeLink in without changing a build file.

## Supported clients

- **Buck2** — point `buck2_remote_execution.engine.address` at your cluster.
- **Reclient** — set `RBE_service` to your NativeLink endpoint.
- **Pants** — configure `[remote_caching]` and `[remote_execution]` in
  `pants.toml`.
- **CMake via [recc](https://github.com/bloomberg/recc)** — wrap compile
  invocations with `recc` and let NativeLink handle the cache layer.

# NativeLink Kubernetes deployments

For most users, the recommended way to deploy NativeLink is via the official
[Helm chart on Artifact Hub](https://artifacthub.io/packages/helm/nativelink/nativelink).

This directory contains supplementary Kustomization building blocks that can
serve as reference points for advanced customization beyond what the Helm chart
provides.
